# Smart Contract and Dapps

Minicurso para o [Sinform 2022](http://sinform.tecnojr.com.br/)

Não esqueça de deixar seu star :star: no repo :grin:

## Rodando o dapp

Instale as dependências

```sh
$ npm install
```

ou

```sh
$ yarn
```

Rode o projeto

```sh
$ npm start
```

ou

```sh
$ yarn start
```
